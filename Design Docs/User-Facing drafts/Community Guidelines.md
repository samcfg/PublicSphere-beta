# ***PublicSphere SourceExchange Moderation Policy***

## ***Moderator Powers***

### ***Standard Powers (Enabled by Default)***

* ***Delete Comments**: Remove comments that violate community standards*  
* ***Lock Threads**: Stop further discussion on problematic threads*

### ***Elevated Powers (Admin Assignment Required)***

* ***Ban Users**: Suspend user accounts from the platform*  
* ***Remove Sources**: Delete source submissions entirely*

## ***Community Standards***

### ***Content Violations***

***Can Be Deleted:***

* *Personal attacks, harassment, or threats*  
* *Spam, excessive self-promotion, or commercial solicitation*  
* *Content unrelated to the source discussion (optional)*  
* *Deliberately false or misleading information* 

***Lock threads:***

* *Discussion becomes consistently hostile*  
* *Thread repeatedly goes off-topic despite warnings*  
* *Source material violates copyright or legal issues*

*Specific articlesâ€™s forums have more strict content guidelines at the request of authors.* 

### ***User Suspension Guidelines***

***Suspend users for:***

* *Repeated comment violations after warnings*  
* *Creating multiple accounts to evade moderation*  
* *Harassment via private messages or across multiple threads*  
* *Posting illegal content*

***Note:** Suspended users can still log in to access account settings and submit appeals, but cannot view any restricted content.*

## ***Moderation Procedures***

### ***Taking Action***

1. ***Always provide a reason** \- Required for every action taken*  
2. ***Add notes** when the situation needs context for other moderators*  
3. ***Check moderation logs** before acting to see if the issue is already being handled*

### ***Documentation Examples***

* ***Delete Comment Reason:** "Personal attack against another user"*  
* ***Lock Thread Reason:** "Discussion repeatedly became hostile despite warnings"*  
* ***Ban User Reason:** "Third violation for harassment after previous warnings"*  
* ***Remove Source Reason:** "Source contains copyrighted material without permission"*

### ***Source Removal***

* ***Use sparingly** \- Sources represent user effort and research*  
* ***Consider rollback first** if source has been edited inappropriately*  
* ***Remove entirely** only for copyright violations, spam, or completely inappropriate content*

## ***Coordination***

### ***Working with Other Moderators***

* ***Any moderator can reverse any action** if they disagree*  
* ***Check with others** via provided communication channel before major actions (bans, source removal)*  
* ***Document disagreements** in action notes for transparency*

### ***Appeals Process***

* *Users appeal via \[contact email address\]*  
* *Any moderator can handle appeals except for their own actions*  
* ***If reversing an action:** Add explanation in notes field*  
* ***If denying appeal:** Respond via email with reasoning*

## ***Getting Help***

* *Questions about policy interpretation*
